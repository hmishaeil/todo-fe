Keep track of your todo tasks

The ToDo application is a sample web application to add, edit, or delete a todo task.

Development Stack
  • Java programming language
  • Spring Boot Framework
  • Angular Framework
  • Bootstrap
  • Amazon SES, S3, RDS, EC2

Features
  • Users can sign up, then they will receive an email to verify their email.
  • Users can login or logout
  • Users can reset their password.
  • Users can create, edit, or delete a todo task.
  • Admin users can add/edit a user. In that case user will receive an email containing temp pass.
  • Support users can only view the users.
